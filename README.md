# Wedding Organizer

## How to run

### For a precompiled release go to [releases](https://github.com/LuckyOx04/WeddingTableOrganizer/releases/tag/v0.5.0-alpha)

### *IMPORTANT:* For source code compilation .Net 10  SDK is needed.

1. Clone this repository
2. Go to the UserInterface folder containing the .cproj file
3. Type `dotnet run`
